DataBase Component
====================

A lightweight database layer on top of PHP's PDO.


## Installation

My Project can be installed with [Composer](http://getcomposer.org)
by adding it as a dependency to your project's composer.json file.

```json
{
    "require": {
        "fobiaweb/database": "*"
    }
}
```

{
    "suggest": {
